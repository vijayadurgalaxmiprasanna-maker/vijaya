Welcome to AWS DevOps Project

This project demonstrates hosting a static website on Amazon S3 with automated deployment using GitHub Actions CI/CD Pipeline.

Project Overview


This project integrates AWS Cloud Computing and DevOps practices. Whenever code is pushed to GitHub, GitHub Actions automatically deploys the latest website files to Amazon S3.
